Backup for my niri + arch dotfiles :)

## Requirements
### Stow
```
pacman -S stow
```

### niri
```
pacman -S niri
```

### fastfetch
```
pacman -S fastfetch
```

### kitty
```
pacman -S kitty
```

### tofi
```
yay -S tofi
```

### waybar
```
pacman -S waybar
```

## Installation

Clone this repo into ~/.dotfiles

```
$ cd ~
$ git clone git@github.com:vewe7/niri-dotfiles.git .dotfiles
$ cd .dotfiles
```

Create symlinks
```
$ stow .
```
